# Financial Performance Dashboard
Designed an interactive Financial Performance Dashboard in Power BI to analyze revenue, profit, and budget variance using KPIs, trend analysis, and dynamic filters.

## Problem Statement
This dashboard is designed to monitor and analyze an organizationâ€™s financial performance by comparing actual values against budgeted figures. It helps stakeholders track revenue, profitability, and variance across sectors, companies, categories, and expense types.

## Tools Used
- Power BI / DAX
- Excel / CSV
- Basic financial modeling concepts

## Key KPIs Shown 
- Total Revenue vs Budget Revenue
- Revenue Variance
- Gross Profit vs Budget Gross Profit
- Operating Profit vs Budget Operating Profit
- Net Profit vs Budget Net Profit

## Dashboard Features 
- KPI cards for quick financial overview
- Revenue, gross profit, operating profit, and net profit analysis
- Budget vs actual variance tracking
- Interactive slicers: Month, Sector, Company, Category, Type
- Revenue trend analysis over time
- Sector-wise and company-wise revenue distribution
- Revenue variance analysis by expense type

## Insights Generated
- Identified revenue shortfalls against budget targets.
- Highlighted sectors contributing the highest revenue.
- Detected cost types causing negative variance.
- Enabled quick comparison of actual vs budget performance.

## How to Use
- Download the .pbix file
- Open it in Power BI Desktop
- Use slicers to filter by month, sector, company, or category
